# Stage 1 Kickoff Checklist

## Hour 1 Setup
- Fill all files in `context/`
- Expand `context/sample-tickets.json` to 50+ records
- Confirm channel values are exactly: `email`, `whatsapp`, `web_form`

## Hours 1-3: Discovery
- Run Prompt 1 from `prompts/stage1-day1-prompt-pack.md`
- Log findings in `specs/discovery-log.md`
- Capture at least 10 cross-channel patterns

## Hours 4-8: Core Prototype
- Run Prompts 2-6
- Ensure channel-aware formatting exists
- Add safe handling for pricing/refund/legal intents

## Hours 9-12: Memory + State
- Run Prompt 7
- Validate follow-up behavior across channels
- Track sentiment, topics, status, channel switch

## Hours 13-16: MCP + Skills
- Run Prompts 8-9
- Define and test at least 5 tools
- Draft final `specs/customer-success-fte-spec.md`

## Exit Gate (Must Pass)
- Working prototype
- Discovery log complete
- Spec drafted
- MCP tools working
- Skills manifest present
- Edge cases documented
